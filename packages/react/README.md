## Quick Start

Step 1: Create a new project with `assistant-ui` pre-configured:

```sh
npx assistant-ui@latest create my-app
cd my-app
```

Step 2: Update the `.env` file with your OpenAI API key. 

Step 3: Run the app:

```sh
npm run dev
```
