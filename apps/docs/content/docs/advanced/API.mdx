## Hooks

### `useAppendMessage`

Appends a message to the current thread.

```tsx
import { useAppendMessage } from "@assistant-ui/react";

const CustomButton = () => {
  const append = useAppendMessage();

  const handleClick = () => {
    append("hello world");
  };

  return <button onClick={handleClick}>Send</button>;
};
```

### `useSwitchToNewThread`

Starts a new thread with an empty chat history.

```tsx
import { useSwitchToNewThread } from "@assistant-ui/react";

const ClearChatButton = () => {
  const switchToNewThread = useSwitchToNewThread();

  const handleClick = () => {
    switchToNewThread();
  };

  return <button onClick={handleClick}>Clear chat</button>;
};
```
