The recommended way to get started is with the following stack:

- NextJS
- Vercel's `ai` package
- `tailwindcss` for styling
- `shadcn/ui` for UI components


<Steps>
### Create a Next.JS project

```sh
npx create-next-app@latest my-app
cd my-app
```

### Initialize `shadcn-ui`

```sh
npx shadcn-ui@latest init
```

### Add `assistant-ui`

```sh
npx @assistant-ui/shadcn@latest add thread
```

This adds a `/components/ui/assistant-ui/thread.tsx` file to your project, which you can adjust as needed.

### Use it in your application:

`/app/index.tsx`

```tsx
"use client";

import { VercelAIAssistantProvider } from '@assistant-ui/react';
import { Thread } from '@/components/ui/assistant-ui/thread';
import { useChat } from 'ai'; 

export default function Home() {
  const chat = useChat();
  return (
    <div className="h-full">
      <VercelAIAssistantProvider chat={chat}>
        <Thread />
      </VercelAIAssistantProvider>
    </div>
  );
}
```

`/app/api/chat/route.ts`

```tsx
import { openai } from '@ai-sdk/openai';
import { streamText } from 'ai';

export const maxDuration = 30;

export async function POST(req: Request) {
  const { messages } = await req.json();

  const result = await streamText({
    model: openai('gpt-4o'),
    messages,
  });

  return result.toAIStreamResponse();
}
```
</Steps>

### Manual Installation

If your stack is not compatible with the `@assistant-ui/shadcn` starter template, you can install `@assistant-ui/react` manually. 
You can check out the source code of the starter template as a reference implementation.

```sh npm2yarn
npm install @assistant-ui/react
```
