---
title: ContentPartPrimitive
description: A part of a message's content. Content parts may be text, image, tool call or UI elements.
---

## Anatomy

```tsx
import { ContentPartPrimitive } from "assistant-ui";

```

## API Reference

### Provider

The context provider for the content part. This is usually handled by `<MessagePrimitive.Content />` so you do not need to use this component directly.
