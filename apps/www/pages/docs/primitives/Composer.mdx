---
title: Composer
description: The user interface to add new messages or edit existing ones.
---

## Anatomy

```tsx
import { ComposerPrimitive } from "assistant-ui";

// creating a new message
const Composer = () => (
  <ComposerPrimitive.Root>
    <ComposerPrimitive.Input />
    <ComposerPrimitive.Send />
  </ComposerPrimitive.Root>
);

// editing an existing message
const EditComposer = () => (
  <ComposerPrimitive.Root>
    <ComposerPrimitive.Input />
    <ComposerPrimitive.Send />
    <ComposerPrimitive.Cancel />
  </ComposerPrimitive.Root>
);
```

## API Reference

### Root

Containts all parts of the Composer.

### Input

The text input field for the user to type a new message.

### Send

The button to send the message.

### Cancel

Cancel the composer. Only available when editing a message.
