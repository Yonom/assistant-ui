---
title: MessagePrimitive
description: A single message in a conversation. Messages may consist of multiple parts.
---

## Anatomy

```tsx
import { MessagePrimitive } from "assistant-ui";

const UserMessage = () => (
  <MessagePrimitive.Root>
    User: <MessagePrimitive.Content />
    
    <BranchPicker />
    <ActionBar />
  </Message.Root>
);

const AssistantMessage = () => (
  <MessagePrimitive.Root>
    Assistant: <MessagePrimitive.Content />
    
    <BranchPicker />
    <ActionBar />
  </Message.Root>
);
```

## API Reference

### Root

Containts all parts of the Composer.

### InProgress

The indicator to show while an assistant message is being generated.  

The contents of this element will be portaled to `<ContentPartPrimitive.InProgressIndicator />`, which allows the indicator to be displayed at the very end of the message.  

### Content

The content of the message in plain text.

### If

Renders children if a condition is met.

```tsx
<Message.If user>
  {/* rendered if message is from the user */}
</Message.If>
<Message.If assistant>
  {/* rendered if message is from the assistant */}
</Message.If>
<Message.If editing>
  {/* rendered if message is being edited */}
</Message.If>
```
